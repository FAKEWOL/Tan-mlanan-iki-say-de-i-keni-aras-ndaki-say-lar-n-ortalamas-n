# Tan-mlanan-iki-say-de-i-keni-aras-ndaki-say-lar-n-ortalamas-n

## Description
This repository calculates the average of numbers from a given number down to 1.

## Features
- Calculates the average of numbers in a specified range.
- Handles any positive integer input for the starting number.

## Installation
No installation is required. Simply clone or download the repository and run the script.

## Usage
To use this script, simply call it with the desired starting number as an argument. For example:

```
php tan-malan-iki-say-de-i-keni-aras-ndaki-say-lar-n-ortalamas-n.php 20
```

This will calculate and output the average of numbers from 20 down to 1.
